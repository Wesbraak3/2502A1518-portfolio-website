# Portfolio Website

This repository contains the source code for my personal portfolio website.

The website is built with **Next.js** and is designed to present who I am, showcase my skills, and highlight the projects I have worked on.

## Overview

The goal of this project is to create a clean and interactive platform where visitors can learn more about me and explore my work.

The site focuses on:
- Presenting my background and skills
- Showcasing personal and professional projects
- Providing a visually clear and structured experience

## Tech Stack

- **Next.js** – Frontend framework
- **Atomic Design** – Component architecture
- **MongoDB** – Database for dynamic data
- **C++ API** – Backend service used to interact with the database

## Architecture

The frontend follows the **Atomic Design methodology**, organizing UI components into reusable layers (atoms, molecules, organisms, etc.).  
Dynamic content is served through a **C++ API**, which communicates with a **MongoDB database**.

## Purpose

This project serves as:
- A personal portfolio
- A playground for experimenting with new technologies
- A place to document and present my projects
